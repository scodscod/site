## Проектная деятельность

Проектная (учебная) практика проводилась в связке с выполнением проекта **«Живой стилистический словарь русского языка. Лингвистический интернет-сервис»** по дисциплине «Проектная деятельность».

В рамках практики был разработан сайт, включающий 5 основных страниц: Главная, О проекте, Участники, Журнал, Ресурсы. Реализовано единое навигационное меню, стилизованное в цветах проекта. Все элементы интерфейса получили скругление углов (12px) и продуманную систему отступов.

Помимо вёрстки, в раздел «Журнал» были добавлены новостные записи, отражающие ключевые этапы разработки: внедрение визуального редактора текста, исправление бага сохранения строк, запуск API для генерации видео, страница работы с JSON-сценариями, а также система авторизации через VK ID с административной панелью. Ведётся поддержка страницы участников и ресурсов.

**Руководитель проекта:** Векшин Георгий Викторович
